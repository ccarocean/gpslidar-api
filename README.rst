Web server for receiving and analyzing LiDAR and GPS Data
=========================================================

Title: gpslidar-api

Options
-------

Installation
------------
Create virtual environment:

.. code-block::

    python -m venv --prompt=api .venv
    source .venv/bin/activate

Inside Virtual Environment:

.. code-block::

    setup.py install


How to run
----------
Source Virtual Environment:

.. code-block::

    source .venv/bin/activate

Run as root:

.. code-block::

    sudo gpslidar-api
